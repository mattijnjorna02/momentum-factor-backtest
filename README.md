# momentum-factor-backtest
Python implementation of a cross-sectional equity momentum strategy

#Strategy

- Universe: US equities
- Signal: 12-month momentum
- Portfolio: Long top 30%, short bottom 30%
- Data source: Yahoo Finance

#Metrics

The strategy evaluates:

- cumulative returns
- volatility
- Sharpe ratio
